## 2. 数据库设计

### 2.1 涨停池历史表 `zt_pool_history`

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PRIMARY KEY | 自增ID |
| stock_code | TEXT | 股票代码 |
| stock_name | TEXT | 股票名称 |
| date | TEXT | 日期 (YYYY-MM-DD) |
| close_price | REAL | 收盘价 |
| change_pct | REAL | 涨幅% |
| seal_amount | REAL | 封板资金(万元) |
| seal_count | INTEGER | 封单量(手) |
| explosion_count | INTEGER | 炸板次数 |
| continuous_days | INTEGER | 连续涨停天数 |
| sector | TEXT | 所属板块 |
| sector_change_pct | REAL | 板块涨幅 |
| super_net_inflow | REAL | 超大单净流入(万元) |
| big_net_inflow | REAL | 大单净流入(万元) |
| main_net_inflow | REAL | 主力净流入(万元) |
| created_at | TEXT | 记录时间 |

### 2.2 板块信息表 `sector_info`

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PRIMARY KEY | 自增ID |
| sector_name | TEXT | 板块名称 |
| date | TEXT | 日期 |
| change_pct | REAL | 板块涨幅 |
| zt_count | INTEGER | 涨停股数量 |
| updated_at | TEXT | 更新时间 |

## 3. API端点设计

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/zt_pool/today` | GET | 获取今日涨停池 |

## 5. 核心功能MVP

### 5.1 定时任务 (每分钟)
- 抓取涨停池 `get_zt_pool()`
- 批量获取资金流向 `get_money_flow_batch()`
- 获取板块数据 `get_sector_ranking()`
- 存入SQLite
- WebSocket推送到前端

### 5.2 Dashboard统计
- 今日涨停总数
- 炸板数 + 炸板率
- 昨日涨停今日表现（昨日涨停今日红盘数）
- 热门板块TOP3

### 5.3 表格筛选/排序
- 按连板数筛选
- 按封板资金筛选
- 按板块筛选
- 支持多列排序
